This section provides an overview of the geometry module, which includes classes and functions for working with soil profiles.

class SoilProfile(soils, **kwargs)[source]

Bases: object

Initialize a SoilProfile object with a list of Soil objects and optional parameters. This class represents a profile of soils, allowing for the calculation of vertical stress

soils

A list of Soil objects representing the different soil layers in the profile.
